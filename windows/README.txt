المتطلبات:
──────────
• Windows 11
• Node.js 24 أو أحدث → https://nodejs.org/
• FFmpeg (للمعالجة الصوتية/المرئية) → https://ffmpeg.org/download.html
  - بعد التحميل: أضف مجلد bin إلى متغير PATH في النظام

خطوات التشغيل (أول مرة):
──────────────────────────
1. انقر مزدوجاً على install.bat
2. بعد اكتمال التثبيت، انقر مزدوجاً على start.bat
3. افتح المتصفح على: http://localhost:3000
4. ادخل إلى "الإعدادات المتقدمة" وأضف:
   - توكن بوت تيليغرام (من @BotFather)
   - مفتاح Gemini AI (من Google AI Studio)

التشغيل بعد التثبيت:
──────────────────────
• انقر مزدوجاً على start.bat
  أو
• انقر بالزر الأيمن على start.ps1 ثم "Run with PowerShell"

تغيير المنفذ:
──────────────
افتح ملف .env وغيّر:
  PORT=3000
إلى أي منفذ تريده مثل:
  PORT=8080

استخدام البروكسي:
──────────────────
افتح ملف .env وأضف:
  HTTPS_PROXY=http://user:password@proxy-host:port

أو للـ SOCKS5:
  SOCKS5_PROXY=socks5://proxy-host:port

التطبيق يفحص الإنترنت تلقائياً:
  ✅ إذا وجد بروكسي ويعمل → يستخدمه
  ✅ إذا لا يوجد بروكسي → يتصل مباشرة
  ❌ إذا لا يوجد اتصال → يُبلّغ بالخطأ

الملفات المهمة:
───────────────
  dist/          ← ملفات الخادم والواجهة المجمّعة
  .env           ← إعدادات البيئة (PORT، PROXY، إلخ)
  start.bat      ← تشغيل على Windows
  start.ps1      ← تشغيل عبر PowerShell
  install.bat    ← تثبيت الحزم (مرة واحدة فقط)
